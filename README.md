# Face Mask Detection using EfficientNetB0

## Project Overview

This project develops a deep learning-based computer vision system for detecting whether a person is wearing a face mask or not. The model performs binary image classification using the EfficientNetB0 transfer learning architecture.

## Dataset

Dataset Source:
Face Mask Detection Dataset

## Project Workflow

1. Import required libraries
2. Download dataset from Google Drive
3. Perform data preprocessing
4. Visualize sample images
5. Build EfficientNetB0 model
6. Train the model
7. Evaluate model performance
8. Generate accuracy and loss graphs
9. Create classification report
10. Generate confusion matrix
11. Test on new images
12. Save trained model
13. Download trained model
